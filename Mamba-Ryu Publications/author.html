Shodai Sennin James A. Overton-Guerra

The Mision of the MAMBA-Ryu Master

I come to be
So you learn how to be
For that which must be
even when
You can no longer be.

I come to raise
So you learn how raise
For that which must be raised
even when
You can no longer raise.

I come persist
So you learn how persist
For that which must persist
even when
You can no longer persist.

I come to remain
So you learn how to remain
For that which must remain
even when
You can no longer remain.

I come to be
So you learn how to be
For that which must be
even when
You can no longer be.
